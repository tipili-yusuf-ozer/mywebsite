My Public Projects

school_bot

Attending lessons to not fail the attendancy requirements is a daunting task, especially when you can code instead of joining them. At least thats how I thought when creating this project. It joins the lessons when it's time and exits them when theres too few people, presumably AFKs or other bots, left in the call. It consists of a C program and a Python script. Python script(the joiner) performs the browser automation with help of the selenium framework. C program (the manager) is started up on computer wake up to check the schedule to either tell joiner to join the live lesson or to sleep until next lesson starts. Fun fact, joiner is written for lessons i took during that period but it can be easily modified to join any lesson by reprogramming the part of the script finds the call link.
